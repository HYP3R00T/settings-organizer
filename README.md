Settings Organizer is a simple yet powerful VSCode extension designed to keep your settings.json file clean and organized. It categorizes settings based on common patterns, ensuring your configuration file remains neat and easy to navigate. No more messy settings.json filesâ€”keep your workspace tidy and efficient with just a single command!

## Features

- Sort global `settings.json` of Visual Studio Code categorically.
- Sort local `settings.json` of Visual Studio Code categorically.

## Usage

- Open the command palette (Ctrl+Shift+P on Windows and Linux, Cmd+Shift+P on OS X) and search for `Settings Organizer`.
- Select the `global` or `local` option based on which file you want to reorganize.
